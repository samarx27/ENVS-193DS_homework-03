# ENVS-193DS_homework-03
This repository was created to complete HW 03 for Env S 193DS and contains data visualizations and analyses for three different data sets and papers. 

What is this repo for? This repository contains a statistical critique, an affective visualization, an example of a test of the strength of a relationship between two independent variables, and figures displaying data. 

Whose repo is it? Who is contributing to it? This is my personal repo. I am the sole contributor. 

What files are there? All code files are under the code folder, these are .qmd files that contain models for personal data and salinity data and correlation tests (Homework03.qmd). The datasets used are in the data folder. 

What code is there, and what does it do? The files include code for creating different models, checking conditions of data sets, and correlation tests.
